# min-cap

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.3.5. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.

capacitor environment was created with `cap add android`.

bun run devcontainer -- -c "node ./src/android/setup.js"

bun run devcontainer -- -c "node ./src/android/build.js"
